Phase-by-phase deliverables

Phase 1: Days 1-30 Foundation & Technical SEO

Objective: Fix the technical SEO basics across every page in scope so Google can crawl, parse, and rank the site properly. By the end of Phase 1, your traditional SEO score should move up to a B-grade foundation.

Deliverables

  1. Meta descriptions, rewrite every page's meta description to land in the 120 to 160 character sweet spot so the snippet that appears in search results actually sells the click instead of being auto-generated by Google.
  2. H1 headlines, add one clear, descriptive H1 headline to every page so Google and AI engines instantly know what each page is about. This is the single highest-impact technical fix on most sites.
  3. Page content depth, expand the thinnest pages with the information visitors and prospects actually want (hours, services, qualifications, results) until every primary page passes the 300 word threshold Google uses to decide a page is worth ranking.
  4. Title tags, rewrite every page title to the 30 to 60 character range Google rewards, with the primary keyword up front and the brand at the end. Same SEO weight as the H1 with the added benefit of showing in browser tabs.
  5. Image alt text, add descriptive alt text to every image so screen readers and Google's image search can identify what's actually in each photo. Required for accessibility and a quiet SEO boost.

Success measures

  • Every page has a unique, descriptive title tag in the 30 to 60 character range.
  • Every page has a meta description in the 120 to 160 character range.
  • Every page has one clear H1 headline that matches search intent.
  • Page-level schema markup live on every primary page.
  • Re-audit confirms SEO pillar at B-grade or higher.
Phase 2: Days 31-60 Answer Engine Optimization

Objective: Layer Answer Engine Optimization on top of the technical foundation. Voice assistants, Google's AI overviews, and featured snippets pull from structured content. Phase 2 makes your site the kind of source they can confidently quote.

Deliverables

  1. Page schema markup, add JSON-LD structured data to every page (Service, Product, Article, or LocalBusiness depending on the page type) so search engines and AI engines can parse the content semantically instead of guessing.
  2. dateModified markers, add dateModified markers to every page so Google knows when content was last touched. Critical for "freshness" ranking signals on industries where current information matters.
  3. Question-style headings, rewrite top-of-page headings as questions a real prospect would type into Google or ChatGPT. This is the single biggest unlock for showing up in featured snippets and AI overviews.
  4. Review schema, add AggregateRating schema to the homepage and testimonials so star ratings appear directly in Google search results, which dramatically lifts click-through rates.

Success measures

  • FAQPage schema on every page that includes question-style content.
  • Question-style headings on the homepage, services pages, and top blog content.
  • AggregateRating schema live and validated.
  • Re-audit confirms AEO pillar at B-grade or higher.
Phase 3: Days 61-90 Generative Engine Optimization & Handoff

Objective: Make the site discoverable and citable by ChatGPT, Claude, Perplexity, and Gemini. Generative engines work differently from Google. Phase 3 closes that gap and hands the system off so you can keep the score where we put it.

Deliverables

  1. llms-full.txt build, write a comprehensive llms-full.txt at the site root that packages every page of long-form content into a single file AI engines can ground their answers in. This is what makes ChatGPT, Claude, and Perplexity confidently cite the business by name.
  2. Organization schema, add sitewide Organization schema with the business name, address, phone, hours, social profiles, and logo so every AI engine knows exactly who they are talking about when the site comes up in a search.
  3. Speakable schema, add Speakable schema to the homepage and top pages so voice assistants (Siri, Alexa, Google Assistant) know which sentences to read aloud when answering a voice query about the business.
  4. Author schema, add Person schema with author bylines on every blog post and content page so AI engines tie content back to the actual humans behind it, which boosts the E-E-A-T signals Google rewards.

Success measures

  • llms.txt and llms-full.txt live at the site root and validated.
  • All major AI crawlers explicitly allowed in robots.txt.
  • Sitewide Organization schema published.
  • Final A-grade audit certifies the turnaround.
  • Handoff documentation delivered so you can maintain the score in-house.
